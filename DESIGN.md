# Agent Index Design System

## Design Read

Agent Index 是一个高密度的产品情报工具。设计应让读者在办公室环境中长时间阅读、检索和核验资料，视觉感受安静、可信、敏捷。

## Dials

- Design variance: 5
- Motion intensity: 3
- Visual density: 6

## Color

- 使用中性的浅色工作区作为默认主题，支持系统深色模式。
- 青绿色是唯一强调色，只用于当前状态、主要行动和真实数据提示。
- 不使用装饰渐变、彩色能力评分或没有数据依据的视觉编码。

## Typography

- 使用系统无衬线字体，优先保证中文与英文混排的稳定性。
- 正文基础字号 16px，紧凑元数据不低于 12px，移动端正文不低于 14px。
- 标题字距不低于 -0.04em，正文行宽控制在 75ch 内。

## Layout

- 最大内容宽度 1280px。
- 首页使用不对称双栏介绍核心价值和真实能力覆盖数据。
- 产品索引与时间线使用固定筛选区加可扫描内容区。
- 详情页使用主内容加辅助事实栏，并在窄屏退化为单栏。

## Components

- 卡片圆角 10px，控件圆角 8px，标签使用小型圆角，不使用大胶囊卡片。
- 卡片以边界、间距和背景分组，阴影仅用于悬浮反馈且模糊不超过 8px。
- 按钮高度至少 44px，具备 hover、active 与 focus-visible 状态。
- 状态点只用于“每周自动更新”这一真实状态。

## Motion

- 仅使用轻微的悬浮位移与颜色过渡。
- 不使用循环装饰动画。
- 尊重 `prefers-reduced-motion`。
