# Agent Index

## Register

Product

## Users

- 需要持续跟踪 Agent 产品变化的产品经理、战略研究者与管理者
- 希望快速判断竞争方向，而不是浏览日常新闻噪音的读者
- 需要核验产品官网、定价与重大更新信源的研究人员

## Product Purpose

维护一个持续更新、可核验的 Agent 产品情报库。网站将产品放进统一坐标系，保留足以改变竞争判断的重大更新，并让读者快速进入原始信源。

## Brand Personality

清晰、严谨、克制、可信、持续更新。像一份经过编辑的研究工作台，而不是新闻流、营销落地页或终端界面。

## Anti-References

- 通用 AI SaaS 营销页
- 终端拟态与装饰性数据大屏
- 充满日常噪音的资讯聚合页
- 依赖虚假评分、进度条或未经解释百分比的分析页面

## Design Principles

1. 判断优先：先帮助读者理解变化意味着什么，再展示原始信息。
2. 证据可见：官网、定价、更新时间与重大动态信源应容易找到。
3. 高密度但可读：允许承载大量信息，但正文、控件与层级必须清晰。
4. 自动化透明：明确说明更新频率、审核标准与数据边界。
5. 渐进披露：索引适合扫描，详情页承载完整拆解与原始记录。

## Accessibility

- 正文与交互元素达到 WCAG AA 对比度
- 支持键盘焦点、触控尺寸和减少动态效果偏好
- 颜色不作为唯一的信息表达方式
- 移动端正文不小于 14px，并避免横向溢出
